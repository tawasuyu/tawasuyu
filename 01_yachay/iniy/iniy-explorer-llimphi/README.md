# iniy-explorer-llimphi

> Llimphi UI of [iniy](../README.md): graph + audit.

Three panels: document tree on the left, assertion graph in the middle, detail of selected assertion on the right (with span of the original, opinions, supports/contradictions, "go to source"). Built so a human can **traverse the full chain of evidence** without opening the raw file.

## Deps

- All `iniy-*` core
- [`llimphi-widget-nodegraph`](../../../02_ruway/llimphi/widgets/nodegraph/README.md), [`tree`](../../../02_ruway/llimphi/widgets/tree/README.md), [`splitter`](../../../02_ruway/llimphi/widgets/splitter/README.md)
