# iniy-explorer-llimphi

> UI Llimphi de [iniy](../README.md): grafo + auditoría.

Tres paneles: árbol de documentos a la izquierda, grafo de afirmaciones en el centro, detalle de la afirmación seleccionada a la derecha (con span del original, opiniones, soportes/contradicciones, "ir a fuente"). Pensado para que un humano pueda **recorrer toda la cadena de evidencia** sin abrir el archivo crudo.
