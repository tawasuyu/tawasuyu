# cosmos-corpus

> Astronomical text corpus of [cosmos](../README.md).

Curated collection of texts (object descriptions, mythology, historical observations) usable as context in "what am I seeing" apps or as dataset for `iniy`. See [GUIA.md](GUIA.md) for the format and curation principles.

## Deps

- [`cosmos-core`](../cosmos-core/README.md), [`cosmos-model`](../cosmos-model/README.md)
- `serde`
