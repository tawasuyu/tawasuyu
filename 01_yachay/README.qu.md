# 01 yachay · yachay

`yachay` (runa-simi: *yachay — riqsiy, yachachiy*). Kaymi **yachay** *cuadrante*: rikuyqa qusqanchikta tinkuwan churaspa, teoría hina kapuq. Simuladorkuna, catalogos, grafu-yachaykuna, hojas-yupachasqa, motor físicokunapas kaypi tiyanku. Imayna `unanchay`pi chaskirqanchik, `yachay`pi tukunchik consultable-projectable estructuraman.

Kay cuadrantepa kamachiyninmi **modelo cheqaq pachawan rikuchikun, manataq sapanmanchu**: shumaq teoría chayqa mana checharsqa, decoración kallan. Sapanka `yachay` aplikacionqa ñawpaq paqarinta sutilla chiqaq-pachawan tinkuy ruwan.

## Aplicaciones

- **[cosmos](cosmos/README.md)** — astronomía cheqaq tupukuywan. Tiempo, ephemerides, coordenadas, WCS, astrología, validación oficial-ephemerideswan.
- **[dominium](dominium/README.md)** — determinista campo-pacha simulador: pichqa físika capakuna (`materia · psique · poder · oro · degradación`) + vector agentes + endógeno ψ↔acción tinkuy.
- **[iniy](iniy/README.md)** — semantik laboratorio. Subjective Logic + subjetividadpa direccion-ejewan willay-auditkuna. Piloto: libro/wiki audits.
- **[nakui](nakui/README.md)** — Excel-laya motor reactivo, fuerte cimientopi: `Decimal` cheqaq, cascada topológica, WAL, ñawpaq pacha-puriy, invariantes atomikos. Kimsa qhawana (matriz · grafu · formulario) kikin token-grafupatamanta.
- **[tinkuy](tinkuy/README.md)** — *tinkuy*: tinkuy, tupay. DOD partikula motor (ECS-SoA + Grid3D + paralela Velocity-Verlet), BLAKE3 snapshots Wawawan tupachisqa.

## Manifesto

> **Yachayqa allinta-pantaymantapas mana manchaymanmi.**
> Modeloqa manan cheqaq hina ranqakhusqachu; herramienta hinan qusqa. Chaninninqa pantakuyninpaq qhawanapi tarikun.
>
> 1. **Determinismo kallpawan, atisparaq.** Kikin muhuyuq, kikin ruwananwanmi — qhipa kutichiyqa hunt'asqanmi.
> 2. **Cheqaqkay ñawpaqman, kusi-ñawi qhipaman.** `Decimal` chaymanta enteros f32-manta ñawpaq, dominioqa chayta munaqtin.
> 3. **Tupukunaqa manan decoracionchu.** SI, IAU, ISO — tukuyninpi sutillapaq; manchanchu "grados-mi" hina musyaylla.
> 4. **Ephemerides, datokuna, hukninkuna simulacionkunawan tinkuchiy.** Sichus hukninkunaqa mana kikinta tarinkuchu, ñawpaq kikiykita tapuy.
