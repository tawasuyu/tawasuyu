# glotona (wawa app)

> Heavy-task eater (batch) in [wawa](../../README.md).

Worker for long jobs (transcoding, compilation, indexing). Receives tasks via [`chasqui`](../../../../02_ruway/chasqui/README.md), reports progress, dumps output to the DAG.
