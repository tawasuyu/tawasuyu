# mudanza (wawa app)

> Migrate between snapshots of [wawa](../../README.md).

Compares two snapshots from [`arje-snapshot`](../../../arje/init/arje-snapshot/README.md), applies the delta. Lets you try a new version and "move in" if you want to keep it, or rollback if not.
