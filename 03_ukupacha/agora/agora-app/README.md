# agora-app

> Llimphi UI of [agora](../README.md).

Thread view + composer + identity. Shortcuts for reply, quote, mark-as-read. No feed algorithm: chronological or per-thread order.

## Usage

```sh
cargo run --release -p agora-app
```

## Deps

- All `agora-*`
- [`llimphi-ui`](../../../02_ruway/llimphi/)
