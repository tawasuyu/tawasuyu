# agora-app

> UI Llimphi de [agora](../README.md).

Vista de hilos + composer + identidad. Atajos para responder, citar, marcar como leído. Sin algoritmo de feed: orden cronológico o por hilo.

## Uso

```sh
cargo run --release -p agora-app
```

## Deps

- Todos los `agora-*`
- [`llimphi-ui`](../../../02_ruway/llimphi/)
