# pineal-render

> Render Llimphi del modelo de [pineal](../README.md).

Convierte una `Scene` de [`pineal-core`](../pineal-core/README.md) en operaciones `vello` dentro de un `SceneCanvas` de Llimphi. Maneja antialiasing, layers compuestas (alpha blending) y exporta a PNG via `pineal-export` cuando se le pide snapshot.

## Deps

- [`pineal-core`](../pineal-core/README.md)
- [`llimphi-ui`](../../../02_ruway/llimphi/) (vello + scene)
