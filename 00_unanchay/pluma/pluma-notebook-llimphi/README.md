# pluma-notebook-llimphi

> Llimphi UI for the [pluma](../README.md) notebook.

Linear render: cell list with inline editor (markdown / code), per-cell "run" button, outputs rendered in-place. Ctrl+Enter runs. Drag-selects like `nada`. Optional sidebar with notebook tree from [`pluma-notebook-store`](../pluma-notebook-store/README.md).

## Deps

- [`pluma-notebook-core`](../pluma-notebook-core/README.md), [`pluma-notebook-exec`](../pluma-notebook-exec/README.md), [`pluma-notebook-store`](../pluma-notebook-store/README.md)
- [`llimphi-ui`](../../../02_ruway/llimphi/) + widgets `text-editor`, `tree`
