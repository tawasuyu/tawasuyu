# pluma-notebook-llimphi

> UI Llimphi del notebook de [pluma](../README.md).

Render lineal: lista de celdas con editor inline (markdown / código), botón "ejecutar" por celda, outputs renderizados in-place. Atajo Ctrl+Enter ejecuta. Drag-selecciona como `nada`. Sidebar opcional con árbol de notebooks de [`pluma-notebook-store`](../pluma-notebook-store/README.md).

## Deps

- [`pluma-notebook-core`](../pluma-notebook-core/README.md), [`pluma-notebook-exec`](../pluma-notebook-exec/README.md), [`pluma-notebook-store`](../pluma-notebook-store/README.md)
- [`llimphi-ui`](../../../02_ruway/llimphi/) + widgets `text-editor`, `tree`
