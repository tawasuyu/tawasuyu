# 00 unanchay · rikuy

`unanchay` (runa-simi: *unanchay, riqsichiy — señalakuy, willakuy*). Kaymi **rikuy** *cuadrante*: imayna willay sistemaman haykun. Tukuy ima hawamanta hamuq monorepoman — qillqa, siq'i, takiy, web, mawk'a kuna, qillqasqakuna —, kaypi haykuspa, hinaspa siguiente cuadrantekunapaq tukurin ukupi mikhuna hina.

Kay cuadrantepa kamachiyninmi **chiqaqtaqa qhawanan, manaraq juzgaspa**: haykuq ima kasqantan k'irikun, ñawpaqman sumaq yachayta `01_yachay`-paq utaq `02_ruway`-paq. Kaypi mana juzgakunchu, chaskinallam.

## Aplicaciones

- **[chaka](chaka/README.md)** — *chaka* (runa-simi: chaka = puente). Mawk'a códigowan, ñawpaq formatowan, sistema rimaynin tinkun.
- **[khipu](khipu/README.md)** — *khipu* (qhipu, watasqa wask'akuna yuyanapaq). Qillqakuna pacha tinkuywan; qunqaytaqmi modelopa ukhunpi: musuq kaqkuna qhipan, mawk'akuna chinkanku.
- **[pineal](pineal/README.md)** — riqsichiy backend mana usuqkuna (cartesiano · polar · mesh · treemap · ...). Sistemapa ñawi.
- **[pluma](pluma/README.md)** — kawsasqa qillqakuna. Markdown átomo-grafu hina; LLM transformadormi, manan autorqa.
- **[puriy](puriy/README.md)** (*puriy*: viajar, recorrer) — web qhawakuq, Servo + Llimphi pataman wiñasqa. Linuxpipas, Wawapipas kaqlla.
- **[rimay](rimay/README.md)** — rimay, simi. Embeddingkuna + verbos.

## Manifesto

> **Ñawpaqtaqa rikuna, hinaqa yuyay.**
> Sistemaman haykuq imamantapas yupaychasqa haykun: mana juzgasqachu, mana huñusqachu, mana wikch'usqachu. Waqaychasqa, contextoyuq, hinaspa ima ruwayta yachaq cuadranteman quykusqa.
>
> 1. **Haykuqpa kikinpa munaynin kanmi.** Ñawpaq kasqannin willaymi — chinkachiymanta hark'akusunchik.
> 2. **Tukuy formato tiyan kuska.** PDF, HTML, BCD, simi, siq'i — mana huñunchu pisi nivel kamalla.
> 3. **Qunqaypas rikuymi.** Mana llapanchu k'iririlla; sistemaqa yachan haykaq kachariyta (khipu).
> 4. **T'ikrayqa chaskiqpa ruwananmi, manan apaqpaqchu.** Ima hina haykun, chayhinan haykun.
