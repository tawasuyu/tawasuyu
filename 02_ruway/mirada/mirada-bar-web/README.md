# mirada-bar-web

> HTML status bar (overlay) of [mirada](../README.md).

Implements [`StatusBar`](../mirada-bar-core/README.md) rendering HTML/CSS over the compositor (via `wlr-layer-shell`). Useful for users who prefer to edit the bar with standard CSS.

## Deps

- [`mirada-bar-core`](../mirada-bar-core/README.md)
- WebView components (`wry` / `servo`)
