# mirada-bar-web

> Status bar HTML (overlay) de [mirada](../README.md).

Implementa [`StatusBar`](../mirada-bar-core/README.md) renderizando HTML/CSS sobre el compositor (vía `wlr-layer-shell`). Útil para usuarios que prefieren editar el bar con CSS estándar.

## Deps

- [`mirada-bar-core`](../mirada-bar-core/README.md)
- Componentes WebView (`wry` / `servo`)
