# supay-scene

> Snapshot del nivel de [supay](../README.md): sectores, mobjs, jugador.

Reconstrucción del estado del nivel desde el motor: lista de sectores con sus pic_idx, mobjs con sprite + posición + ángulo, jugador con cámara. Sin datos de gameplay propios — sólo lo necesario para renderear.

## Deps

- `serde`, `glam`
