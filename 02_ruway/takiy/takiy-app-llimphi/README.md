# takiy-app-llimphi

> Llimphi UI (sequencer + synthesis) of [takiy](../README.md).

Editable pianoroll, mixer, synth presets, transport (play/stop/loop). WAV export.

## Usage

```sh
cargo run --release -p takiy-app-llimphi
```

## Deps

- [`takiy-core`](../takiy-core/README.md), [`takiy-synth`](../takiy-synth/README.md), [`takiy-playback`](../takiy-playback/README.md), [`llimphi-ui`](../../llimphi/)
