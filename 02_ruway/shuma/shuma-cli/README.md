# shuma-cli

> CLI puro de [shuma](../README.md). Sin UI Llimphi.

Modo terminal estándar: lee stdin, escribe stdout. Sirve para uso por SSH o cuando no hay Llimphi disponible.

## Uso

```sh
cargo run --release -p shuma-cli
```

## Deps

- [`shuma-core`](../shuma-core/README.md), [`shuma-line`](../shuma-line/README.md), [`shuma-exec`](../shuma-exec/README.md)
