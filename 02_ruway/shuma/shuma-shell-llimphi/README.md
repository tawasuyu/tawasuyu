# shuma-shell-llimphi

> Llimphi UI shell of [shuma](../README.md).

App on the 4-slot chassis: TopBar (command bar), Main (output stream), BottomBar (status), DrawerTab (Quake drawer). Each slot is a swappable module.

## Usage

```sh
cargo run --release -p shuma-shell-llimphi
```

## Deps

- All `shuma-module-*`, [`shuma-shell-render`](../sandbox/shuma-shell-render/README.md)
