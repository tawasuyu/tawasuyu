# shuma-shell-render

> Renderer de output (ANSI, imágenes, links) de [shuma](../../README.md).

Convierte un `Output` en `View<Msg>`. ANSI escape codes → colores; URLs → links clickables; imágenes inline (PNG/JPEG/SVG).

## Deps

- [`shuma-core`](../shuma-core/README.md), [`llimphi-ui`](../../../llimphi/)
