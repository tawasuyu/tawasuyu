# shuma-shell-render

> Output renderer (ANSI, images, links) of [shuma](../../README.md).

Converts an `Output` to `View<Msg>`. ANSI escape codes → real colors; URLs → clickable links; inline images (PNG/JPEG/SVG).

## Deps

- [`shuma-core`](../shuma-core/README.md), [`llimphi-ui`](../../../llimphi/)
