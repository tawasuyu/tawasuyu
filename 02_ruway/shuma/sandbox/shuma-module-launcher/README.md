# shuma-module-launcher

> Launcher module (DrawerTab) of [shuma](../../README.md).

Quake drawer slot: `F12` deploys it, fuzzy-finder of commands and recent apps. Wraps [`mirada-launcher`](../../../mirada/mirada-launcher/README.md).

## Deps

- [`shuma-module`](../shuma-module/README.md), [`mirada-launcher`](../../../mirada/mirada-launcher/README.md)
